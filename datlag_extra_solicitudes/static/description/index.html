Modulo de mejora de ausencias

Administra  las vacaciones de tus empleados.



Este modulo amplia la Gestion de vacaciones de RR.HH. predeterminada de Odoo con funciones adicionales adaptables para gestionar las vacaciones de los empleados.

Este modulo mejora algunos elementos del modulo de ausencias agregando:



1. Solicitar prima vacacional
2. Solicitar archivo por tipo de vacaciones

Para configurar este modulo agrege la opcion deseada en los tipos de ausencias

tipos de solicitudes



Prima vacacional



Solicitar archivo
